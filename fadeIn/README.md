gOpSlider
========

> 这是一块基于jquery的透明度切换slider插件



**Use**
*  	$('#target').gOpSlider(options);


**options**
* currClass: 'navOn',  //当前数字按钮样式
* eventNav: 'click',   //数字按钮事件
* navId: '#mar3Nav',   //数字按钮存放容器
* time: 3000,          //切换速度
* beforeCallback: function(){}, //执行前回调
* afterBack: function(){}       //执行后回调
